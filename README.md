# ACME Solar Prediction (AMS 2014 Kaggle Competition)

Group project for Machine Learning I (MSc Business Analytics & Data Science).

## Goal
Predict solar energy output using historical measurements and station metadata.

## Files
- `ACME_Prediction_Group_2.ipynb` - full workflow (EDA + modeling)
- `EDA_Visuals_Group2.pptx` - EDA slides
- `Group 2 summary.pdf` - short project summary
- `predictions_acme.csv` - final predictions submission

## Data
Data comes from the Kaggle competition (not included in this repo).
